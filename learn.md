1. **不要忘記import random**  
   1. random.uniform(n1,n2) 得出n1~n2隨機**float**  
   2. random.randint(n1,n2) 得出n1~n2隨機**int**  
   3. random.shuffle(list)  把list*洗亂*  
   4. random.random()       產生 0 到 1 之间的随机**float**  
   5. random.randrange(1,100,2) 1到100的间隔为2的随机 **int** == 產生奇數
